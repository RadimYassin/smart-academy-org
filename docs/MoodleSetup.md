# Moodle Course Guide

This guide explains **how to set up Moodle, create a course, add content, enroll students, and share the course link**. Follow these steps to get your course online and accessible to your team or students.

---

## Step 1: Set up Moodle

You have two main options:

### Option A: Use MoodleCloud (Easiest)
1. Go to [MoodleCloud](https://moodlecloud.com/).  
2. Click **Start Free Site** and create an account.  
3. You will get a URL like `https://yourname.moodlecloud.com`.  
4. Log in as Admin to start creating courses.

**Pros:** Quick, no server needed  
**Cons:** Free version has limitations on users and storage

### Option B: Install Moodle on Your Own Server (Advanced)
1. Install a web server (Apache/Nginx), PHP 8+, and MySQL/PostgreSQL.  
2. Download Moodle from [moodle.org](https://download.moodle.org/).  
3. Follow the official guide: [Installing Moodle](https://docs.moodle.org/en/Installing_Moodle).  

**Pros:** Full control, unlimited users  
**Cons:** Requires technical knowledge and hosting

---

## Step 2: Create a Course
1. Log in as **Admin or Teacher**.  
2. Navigate to: **Site Administration > Courses > Add a new course**.  
3. Fill in the course details:  
   - **Course full name:** e.g., Mathematics 101  
   - **Short name:** e.g., Math101  
   - **Course category:** Optional  
   - **Course format:** Weekly, Topics, or Single activity  
4. Click **Save and display**.

---

## Step 3: Add Course Content
1. Enter the course and click **Turn editing on**.  
2. Add resources and activities using **Add an activity or resource**:  
   - **Lesson:** Interactive lessons for students  
   - **Quiz:** Tests and assessments  
   - **Assignment:** Homework submission  
   - **File/URL:** PDFs, videos, links  
3. Organize content by **weeks** or **topics** for easy navigation.  
4. Optional: Add labels, images, or videos for better presentation.

---

## Step 4: Enroll Students
You can add students manually or allow them to self-enroll.

### Option A: Manual Enrollment
1. Go to **Course Administration > Users > Enrolled users > Enroll users**.  
2. Search for students by name or email.  
3. Click **Enroll**.

### Option B: Self-Enrollment (Recommended)
1. Go to **Course Administration > Users > Enrollment methods > Add method > Self enrollment**.  
2. Set an **Enrollment key** (password students will use to join).  
3. Save changes.  

**Tip:** Self-enrollment allows students to join themselves and is easier for large groups.
